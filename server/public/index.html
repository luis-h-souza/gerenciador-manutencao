Sistema de Gerenciamento de Manutenção (SGM) (1.0.0)

Download OpenAPI specification:

Luis Henrique: luis.h.desenvolvimento@gmail.com URL: https://github.com/luis-h-souza/gerenciador-manutencao

O Sistema de Gerenciamento de Manutenção (SGM) é uma plataforma completa para gerenciamento de tarefas, controle financeiro, estoque, checklists e dashboards executivos.

Características Principais

  • REST API Stateless com JWT + Refresh Tokens
  • RBAC com 6 roles e isolamento regional
  • Rate Limiting em 3 camadas
